## 🎨 設計特點

### 顏色方案

- 主要漸層：紫色到紫羅蘭 (#667eea → #764ba2)
- 背景：白色卡片配漸層背景
- 文字：深灰色主文字，中灰色輔助文字
- 按鈕：漸層填充，懸停時有提升效果

### 互動效果

- 按鈕懸停：向上移動 + 陰影增強
- 輸入框焦點：邊框顏色變化 + 外發光效果
- 標籤切換：淡入動畫
- 表單提交：錯誤訊息震動動畫
- 頁面載入：從下往上滑入動畫

### 響應式設計

- 在小螢幕上自動調整佈局
- 選項區域在手機上改為垂直排列
- 填充和字體大小自適應
